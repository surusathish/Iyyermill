# **Yarn Manufacturing Website**

## **Overview**
This is a professional website designed for a **yarn manufacturing company** to showcase products, attract foreign clients, and provide a seamless user experience. The website is built with **HTML, CSS, JavaScript**, and **Bootstrap**, ensuring responsiveness and modern aesthetics.

## **Features**
- **Home Page**: Introduction to the company's premium yarn manufacturing.
- **About Us**: Company history, mission, and values.
- **Products Page**: Display of yarn varieties with descriptions and specifications.
- **Certifications Page**: Showcasing quality certifications and standards.
- **Contact Page**: Contact form, Google Maps integration, and WhatsApp chat.
- **Video Section**: Embedded video showcasing the manufacturing process.
- **WhatsApp Integration**: Floating WhatsApp button with animation.
- **Optimized UI**: Updated styling and animations for an elegant user experience.
- **Improved Form Handling**: Prevents redirection and displays success/error messages in a non-intrusive way.

## **Technologies Used**
- **HTML5** for structured content
- **CSS3 (Bootstrap & Custom CSS)** for styling and responsiveness
- **JavaScript** for form validation and interactive elements
- **FontAwesome** for icons
- **Google Maps API** for location display

## **Setup & Installation**
1. Clone the repository or download the project files.
2. Ensure all necessary images and assets are placed in the `images/` directory.
3. Open `index.html` in a browser to view the website.
4. Deploy to a hosting provider if needed.

## **Customization Guide**
- **Updating Contact Information**: Modify `index.html` in the **Contact Section**.
- **Replacing Video**: Update the `iframe` source in the **Video Section**.
- **Changing WhatsApp Number**: Modify the `href` in the WhatsApp button link (`index.html`).
- **Styling Adjustments**: Modify `style.css` to tweak colors, spacing, or animations.
